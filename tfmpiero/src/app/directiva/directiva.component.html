Directivas

Las directivas son una de las piezas clave que nos brinda Angular para manipular el DOM de manera más sencilla y por tanto tener código más mantenible. Se puede decir que las directivas son pequeñas instrucciones de código que se aplican sobre el DOM. En cierto sentido, los componentes son un tipo de directiva, ya que cuando creamos un componente lo que estamos diciendo es que se construya el HTML a partir de nuestra clase .ts.
